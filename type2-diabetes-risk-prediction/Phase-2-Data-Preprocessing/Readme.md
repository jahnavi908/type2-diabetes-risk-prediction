# Phase 2 - Data Preprocessing

## 📌 Objective
To clean and prepare the synthetic dataset for machine learning models.

---

## 🔧 Steps Performed

### 1. Data Loading
- Imported synthetic dataset from Phase 1

### 2. Data Cleaning
- Checked missing values
- Verified data types
- Ensured consistency

### 3. Label Encoding
- Converted categorical values (if any) into numeric format

### 4. Feature Scaling
- Applied StandardScaler to normalize features
- Ensures equal contribution of all variables

---

## 📊 Why Scaling is Important

- Improves model performance
- Helps distance-based algorithms (KNN, SVM)
- Prevents feature dominance

---

## 📁 Output

- Cleaned and scaled dataset ready for ML models

---

## 🎯 Outcome

Preprocessed dataset optimized for model training.
