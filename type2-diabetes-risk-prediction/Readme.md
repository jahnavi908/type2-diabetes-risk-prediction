# Type-2 Diabetes Risk Prediction using Machine Learning

## 🧠 Project Overview

This project focuses on predicting the risk of Type-2 Diabetes using machine learning techniques. Early detection of diabetes is crucial to prevent severe complications such as heart disease, kidney failure, and nerve damage.

A synthetic healthcare dataset is created using medical ranges of key health indicators such as:
- BMI (Body Mass Index)
- Age
- Insulin Levels

Multiple machine learning algorithms are applied to classify whether a person is at risk of developing Type-2 Diabetes.

---

## 🎯 Problem Statement

Early diagnosis of Type-2 Diabetes is critical for preventive healthcare. The objective is to:

- Generate a synthetic dataset of 4001 health records
- Perform preprocessing and feature scaling
- Train multiple classification models
- Evaluate performance using confusion matrix and recall-focused metrics

---

## 📊 Project Pipeline

Data Synthesis → Data Preprocessing → Feature Scaling → Model Training → Evaluation
