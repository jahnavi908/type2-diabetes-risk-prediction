# Phase 1 - Data Synthesis (Synthetic Data Generation)

## 📌 Objective
To generate a synthetic healthcare dataset for Type-2 Diabetes risk prediction using medically realistic ranges.

---

## 🧠 Data Source
The dataset used in this project is **synthetically generated using Python (GPT-assisted logic)**.  
It is NOT collected from real patients or hospitals.

This approach is used for:
- Machine learning experimentation
- Safe dataset creation without privacy concerns

---

## 📊 Features Generated

- Age (18–80 years)
- BMI (18–40 range)
- Insulin levels (10–200 µU/mL)
- Glucose levels (70–200 mg/dL)
- Blood Pressure (60–140 mmHg)

## 📁 Output

- Synthetic dataset of 4001 records
- Saved as CSV file

---

## 🎯 Outcome

A structured synthetic dataset ready for preprocessing and machine learning.
