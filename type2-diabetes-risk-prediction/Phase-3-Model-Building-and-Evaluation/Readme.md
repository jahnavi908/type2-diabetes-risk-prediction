# Phase 3 - Model Building & Evaluation

## 📌 Objective
To build and evaluate multiple machine learning models for Type-2 Diabetes risk prediction.

---

## 🤖 Models Used

- Logistic Regression
- Support Vector Machine (SVM)
- Decision Tree Classifier
- Random Forest Classifier
- K-Nearest Neighbors (KNN)
- K-Means Clustering

---

## 📊 Evaluation Metrics

- Accuracy Score
- Confusion Matrix
- Classification Report
- Heatmap Visualization

---

## 📈 Visualizations

- Correlation Heatmap
- Confusion Matrix
- Model comparison charts

---

## 🎯 Key Insight

Models were compared based on accuracy and overall classification performance.

---

## 🏆 Outcome

The best performing model was selected based on evaluation results.
